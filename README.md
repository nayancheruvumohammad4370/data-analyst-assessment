#  Data Analyst Assessment

##  Overview
This project analyzes SFTP connection logs to identify success/failure patterns, error trends, and system behavior.

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

---

##  Dataset
The dataset contains:
- Email (customer identifier)
- Operation type
- Success/Failure status
- Error messages
- Timestamp (created_at)

---

##  Analysis Performed
- Data cleaning and preprocessing  
- Success vs Failure analysis  
- Error pattern identification  
- Customer impact analysis  
- Time-based trend analysis  
- Change point detection  

---

## Key Insights
- Identified major failure patterns and top error types  
- Detected spike in failures during specific time period  
- Found most impacted customers based on failure rate  
- Analyzed operations contributing to failures  

---

##  Assumptions
- `success = False` represents failure  
- Errors are derived from `error_message` column  

---

##  Conclusion
The analysis highlights potential system-level issues and provides insights into failure trends and impacted users. Recommendations include improving monitoring, handling retries, and fixing recurring errors.
